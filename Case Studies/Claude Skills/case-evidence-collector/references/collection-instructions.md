# Spoon-Fed Collection Instructions

Use these instructions verbatim when requesting evidence from the user. Replace `{Company}`, `{Product}`, and `{Year}` with actual values from intake.

---

## Category A: Earnings Calls / Transcripts

**Search queries to run:**
1. `"{Company}" earnings call transcript {Year}`
2. `"{Company}" shareholder letter {Year}`
3. `"{Company}" quarterly results {Year}`
4. `site:seekingalpha.com "{Company}" earnings call`

**What to extract (5-10 bullets per transcript):**
- Growth drivers mentioned by executives
- Customer adoption friction or challenges
- Enterprise vs SMB commentary
- Competitive mentions
- Retention, conversion, or churn references
- Revenue or user count milestones

**Copy-paste template:**
```
### Earnings Call: [Quarter] [Year]
Source type: Earnings call transcript
Title: [e.g., "Q3 2016 Earnings Call"]
Date: [e.g., November 2016]
Verbatim / Paraphrased: [choose one]

• [Bullet 1]
• [Bullet 2]
• [Bullet 3]
```

## Category E: Review Themes (G2 / Capterra)

**Search queries to run:**
1. `"{Product}" reviews G2`
2. `"{Product}" reviews Capterra`
3. `"{Product}" user reviews pros cons`
4. `"{Product}" vs [competitor] reviews`

**What to extract:**
- Top 5 praise themes (what users love)
- Top 5 complaint themes (what users dislike)
- 3-5 adoption blockers (barriers to getting started or scaling)
- Enterprise vs SMB differentiation (if visible in reviews)

## Category F: Industry Benchmarks (Numeric)

**Search queries to run:**
1. `[industry] benchmark report {Year}`
2. `[industry] average [metric] statistics`
3. `"{Product}" market size TAM`
